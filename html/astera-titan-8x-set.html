Astera Titan 8x LED Tube Set



Inventory

1 Available



The Astera Titan 8x LED Tube Set is an advanced lighting solution that delivers vibrant, high-quality lighting for film, photography, and event production. Each of the eight Titan LED tubes features RGB, mint, and amber LEDs for precise color control and a wide color temperature range from 1750K to 20,000K. The Titan Tubes provide exceptional color accuracy, with CRI and TLCI ratings of 96+, making them ideal for professional lighting applications where precision is required.

These wireless, battery-powered LED tubes are designed for ultimate flexibility on set. Each tube can run for up to 20 hours on battery power, and they are fully compatible with the AsteraApp for remote control, allowing you to adjust brightness, color, and lighting effects from a distance. The Titan Tubes also support DMX via the AsteraBox ART7 for professional lighting setups.

The Astera Titan 8x Set includes eight tubes, along with mounting hardware and a robust charging case that allows you to charge all tubes simultaneously. With a durable design and IP65 rating for water and dust resistance, the Titan Tubes are perfect for use in both indoor and outdoor environments, ensuring reliability in all conditions.
