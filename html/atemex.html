Blackmagic ATEM Mini Extreme



Inventory

1 Available



The Blackmagic ATEM Mini Extreme is a powerful live production switcher designed to support more complex, multi-camera streaming setups. Equipped with eight HDMI inputs, multiple outputs, dedicated streaming hardware, and an expanded control panel, this switcher is ideal for large productions, live events, conferences, and studio broadcasts.

With advanced features like two separate HDMI outputs, two USB outputs, two independent media players, and up to 16-window multiview, the ATEM Mini Extreme offers unparalleled flexibility for professional streaming and live mixing. It supports various transitions, DVE effects, graphics overlays, and provides intuitive control through both physical buttons and ATEM Software Control.



Specifications

  • Video Inputs: 8x HDMI (up to 1080p60)
  • Video Outputs: 2x HDMI (Program/Multiview), 2x USB (Webcam Out), Ethernet for direct streaming
  • Audio Inputs: 2x 3.5mm stereo mini jack, embedded HDMI audio from each input
  • Streaming Engine: Hardware H.264 encoder for direct streaming via Ethernet
  • Multiview: Up to 16-window multiview monitoring of all sources, status, and audio levels
  • Other Features: 2 media players, picture-in-picture, keyers, transitions, and more
  • Power: External 12V DC power supply


Key Features

  • 8 HDMI Inputs: Easily switch between multiple cameras, computers, or other HDMI sources.
  • Dual USB Outputs: Simultaneously output to two different USB webcam feeds for various platforms or backup setups.
  • Independent HDMI Outputs: Assign dedicated program, multiview, or clean feed outputs to two different monitors or recorders.
  • Expanded Multiview: Monitor up to 16 windows on a single screen, including sources, preview, program, audio meters, and streaming status.
  • Onboard Control Panel: Large set of physical buttons for quick access to transitions, effects, audio controls, and camera inputs.
  • Full ATEM Software Control: Unlock advanced features, set up macros, load graphics, and manage media pools via Ethernet or USB.
  • Seamless Live Streaming: Stream directly to popular platforms like YouTube, Facebook, or Twitch via built-in hardware encoder.
